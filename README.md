A simple, modern, and secure password generator built with HTML, CSS, and JavaScript. This tool allows you to create strong, random passwords and customize them to your needs.

## Features

*   **Customizable Password Length:** Generate passwords from 8 to 64 characters long.
*   **Character Set Selection:** Include or exclude uppercase letters, lowercase letters, numbers, and symbols.
*   **Secure Generation:** Uses the `crypto.getRandomValues()` API for cryptographically secure random number generation.
*   **Password Strength Meter:** Visual feedback on the generated password's strength, including entropy and estimated crack time.
*   **Batch Generation:** Create multiple passwords at once.
*   **Password History:** View the last 3 generated passwords.
*   **Copy to Clipboard:** Easily copy the generated password.
*   **Save to Vault:** Save your favorite passwords to your browser's local storage.
*   **Export Vault:** Export your saved passwords as a JSON file.
*   **Multiple Themes:** Choose from three professional color themes to suit your preference.

## How to Use

1.  Download or clone this repository.
2.  Open the `secure-password-generator.html` file in your web browser.
3.  Adjust the password length and character sets as needed.
4.  Click the "Generate" button to create a new password.

## Themes

This password generator comes with three beautiful themes:

*   **Midnight:** A sophisticated dark theme with deep blues and vibrant accents.
*   **Matrix:** A classic "hacker" theme with a black background and bright green text.
*   **Daylight:** A clean and modern light theme with excellent readability.

You can switch between themes using the buttons in the top-right corner of the page. Your theme preference will be saved for your next visit.
